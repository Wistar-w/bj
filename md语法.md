# Markdown 常用语法备忘

## 1. 标题
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 2. 引用
> 这是一级引用
>> 这是二级引用
>>> 这是三级引用

## 3. 列表
### 无序列表
- 项目1
+ 项目2
* 项目3

### 有序列表
1. 第一项
2. 第二项
3. 第三项

### 嵌套列表
- 外层项
  - 内层无序项
  1. 内层有序项
- 另一外层项

### 任务列表（Todo）
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项

## 4. 表格
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| a | b | c |
| 单元格 |  居中    |    靠右 |

## 5. 换行与分割线
- 换行：行末添加两个或以上空格，然后回车  
  这是上一行的换行结果
- 分割线：使用三个或更多 `*`、`-` 或 `_`

---

## 6. 文本样式
| 样式 | 代码 |
| :--: | :--: |
| *斜体* | `*斜体*` 或 `_斜体_` |
| **粗体** | `**粗体**` 或 `__粗体__` |
| ***粗斜体*** | `***粗斜体***` |
| ~~删除线~~ | `~~删除线~~` （GFM 扩展） |
| ==高亮== | `==高亮==` （部分平台支持） |
| <u>下划线</u> | `<u>下划线</u>` （HTML 标签） |

## 7. 图片与链接
### 超链接
- 基本用法：`[显示文本](https://example.com "可选标题")`  
  示例：[点击访问示例](https://example.com "示例网站")

- 自动链接：直接使用 `<URL>`  
  示例：<https://www.markdownguide.org>

- 引用链接：先定义引用标识，再使用
